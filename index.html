在大學期間參加了系學會並且擔任了迎新總召與美編，負責活動規劃、資源分配與對外溝通，培養了在時間壓力下推進任務、協調團隊與處理突發狀況的能力。​

參與校內 Library A-thon Python 程式設計與 Arduino 系統工程設計競賽，與團隊合作完成題目實作，我在團隊中主要負責蒐集與整理資料，以及比賽過程中的程式與系統除錯，協助排除感測數據與邏輯錯誤，最終兩項競賽分別獲得季軍與亞軍，累積在高壓時限下協作解題與完成實作的經驗。

大學專題負責 PCB 瑕疵影像分類專案，從資料整理到模型訓練與調整，熟悉實驗規劃與結果分析；研究所則以水下聲音辨識為主題，處理高雜訊、多類別資料，實作過從降噪、特徵設計到比較多種模型（含 KNN、輕量型 CNN、LSTM 等）的流程，最後設計出以 CNN‑LSTM 擷取時頻序列特徵、再結合 MobileNetV3 進行分類的架構，有效提升不同海洋哺乳類聲音的辨識穩定度，強化了獨立研究與系統化思考能力。
